# 💸 UPI Transaction Simulator

🚧 **Status: Under Development**

A backend-driven UPI simulation system built using Spring Boot that allows users to send and receive money via virtual UPI IDs with secure authentication and real-time balance updates.

---

## 🚀 Features

* 💰 Virtual wallet system (send/receive money)
* 🔐 JWT-based authentication & role-based access (User/Admin)
* ⚡ Real-time balance updates
* 🛡️ Fraud detection (duplicate transaction prevention)
* 📊 Daily transaction limit enforcement
* 📜 Transaction history with pagination & filters
* 🔁 ACID-compliant database transactions with rollback support

---

## 🛠️ Tech Stack

* Java (Spring Boot)
* MySQL
* REST API
* JWT Authentication
* HTML, CSS, JavaScript
